#  Expense Analysis & Predictor

##  Project Overview

This project is a Python-based application developed to analyze daily expenses and predict total monthly expenditure.
It uses a CSV file as input and applies basic data analysis along with a simple Machine Learning model.

---

##  Objectives

 Analyze daily expense data
 Calculate total and average expenses
 Visualize expense trends
 Predict monthly expenditure

---

##  Features

 Reads data from CSV file
 Shows total and average expenses
 Displays graph of daily expenses
 Predicts monthly expense using Linear Regression

---

##  Technologies Used

 Python
 Pandas
 NumPy
 Matplotlib
 Scikit-learn

---

project_folder/

##  Dataset Format

The dataset file (`expenses.csv`) contains:

| Day | Expense |
| --- | ------- |
| 1   | 200     |
| 2   | 150     |
| 3   | 300     |

---

##  How to Run

1. Install required libraries:

```
pip install pandas numpy matplotlib scikit-learn
```

2. Place `expenses.csv` in the same folder as `main.py`

3. Run the program:

```
python main.py
```

4. Select option:

 Show Analysis
 Predict Expense
